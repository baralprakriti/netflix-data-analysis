# Netflix Data Analysis Project (SQL + Python)

## Project Overview
This project involved building a data pipeline to analyze Netflix's content library. I moved raw CSV data into a MySQL database and used Python to extract business insights regarding content trends and genres.

##  Tools Used
- **Database:** MySQL Workbench
- **Language:** Python (Pandas, SQLAlchemy)
- **Visualization:** Matplotlib, Seaborn
- **Environment:** VS Code on macOS

## Key Insights
- **Content Strategy:** Found that Netflix's library is heavily focused on **International Movies** (2,752 titles), indicating a global expansion strategy.
- **Production Trends:** Identified that the average Netflix movie length is **99.58 minutes**, which aligns with standard theatrical runtimes.
- **Data Engineering:** Developed a Python script to handle character encoding issues (Latin-1) and automated the data upload to a relational database.
